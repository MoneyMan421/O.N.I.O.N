# Security Policy

## Supported Versions

We support only the latest version.

## Reporting a Vulnerability

Please **DO NOT** open public issues for security bugs.

To report a vulnerability, contact the maintainers directly or use GitHub's private vulnerability reporting feature:
1. Go to the repository's **Security** tab.
2. Click **Report a vulnerability**.
3. Provide a clear description, reproduction steps, and any relevant context.

We will acknowledge reports within 5 business days and aim to resolve confirmed vulnerabilities promptly.
