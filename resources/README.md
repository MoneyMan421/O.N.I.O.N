# O.N.I.O.N Resources

This directory contains diagrams, images, and other visual resources.

## Structure

- `diagrams/` - Architecture diagrams, flowcharts
- `images/` - Logos, icons, screenshots

## Diagram Sources

Diagrams should be created using:
- Draw.io / diagrams.net
- Mermaid (for text-based diagrams)
- PlantUML
- LucidChart

Export as both source format and PNG/SVG for use in documentation.

## Image Guidelines

- Use SVG when possible for scalability
- PNG for screenshots and photos
- Optimize images for web use
- Include alt text in documentation

## Naming Convention

```
{layer}-{component}-{type}.{ext}

Examples:
- l4-pdp-flowchart.svg
- l5-approval-workflow.png
- architecture-overview.svg
- logo-onion.svg
```
