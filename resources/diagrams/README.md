# Architecture Diagrams

This directory stores visual documentation for the O.N.I.O.N platform.

## Included diagrams

- `onion-layered-architecture.mmd` — layered trust and verification model
- `onion-end-to-end-flow.mmd` — GitHub to Azure runtime delivery flow
- `onion-system-blueprint.mmd` — blueprint of runtime services and control planes

## Usage

These Mermaid source files can be:
- rendered directly by supported Markdown viewers,
- embedded in documentation,
- exported to SVG or PNG for presentations and reports.
